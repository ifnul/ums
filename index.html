Welcome to UMS - University Management System.

This project is a part of one big reform, that started in Ukraine after revolution of dignity.It consist of several parts such as backend, admission frontend, etc.. This is a pilot project and it is constantly changing. Our ultimate goal is to replace paper work in all ukrainian universities into digital one.

Project is declared to use Java and Linux. Don't forget to install them.

Technologies

To start with, we use Java as our backend language.We have several frameworks connected with backend, such as Maven, Spring, JPA(Hibernate), Liquibase, Gatling. We have AngularJS on top of our frontend admission part.We love to use tests, we have around 2200 unit tests and 1260 integration tests. We use Jenkins as CI(Continuous Integration) and CD(Continuos Deployment).
